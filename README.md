# Fitness Pro Frontend

Aplicación web enfocada en entrenamiento de gimnasio, seguimiento de progreso y retroalimentación inteligente mediante visión por computadora.

---

## 🚀 Tecnologías Utilizadas

* React
* Vite
* React Router DOM
* Recharts
* CSS Modular / Arquitectura basada en componentes

---

## 📥 Clonar el Repositorio

```bash
git clone git@github.com:GonzaloTT/Fitness_Pro.git
```

Entrar al proyecto:

```bash
cd fitness-pro/frontend
```

---

## 📦 Instalar Dependencias

```bash
npm install
```

---

## ▶️ Ejecutar en Localhost

```bash
npm run dev
```

Después abrir en navegador:

```text
http://localhost:5173
```

---
